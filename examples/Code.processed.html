Code Blocks

Code blocks are enclosed in triple backticks (```) and are used to display code snippets. You can specify the language for syntax highlighting.

```python
def hello_world():
    print("Hello, World!")
```

Produces:

def hello_world():
    print("Hello, World!")

And

```javascript
function helloWorld() {
    console.log("Hello, World!");
}
```

Produces:

function helloWorld() {
    console.log("Hello, World!");
}

Code files custom block

A codefile (included in the template) block is available to add a file name to the code block.

::: codefile | Project/Program.cs
```csharp
using System;
namespace HelloWorld
{
    class Program
    {
        public static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}
```
:::

Custom Syntax Highlighting

Different styling can be applied by updating the code-github-dark.css file in the .chancellor folder. The template uses github-dark as the default style.

You may update the CSS classes for different tokens, or browse highlight.js for examples by selecting a style and copying the content to the code-github-dark.css file.
